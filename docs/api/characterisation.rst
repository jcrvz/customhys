Characterisation
================

.. automodule:: customhys.characterisation
   :members:
   :undoc-members:
   :show-inheritance:

   .. warning::

      This module is **work-in-progress**. The API may change in future releases.

   The ``Characteriser`` class provides metrics for characterising benchmark function
   landscapes through sampling strategies (e.g., Lévy walks) and kernel density
   estimation.
